# @guwidoe/pi-wsl-clipboard-image

WSLg clipboard image extension for pi.

## Features

- `Alt+V` shortcut to paste clipboard image into the prompt as `@/tmp/...png`
- `paste-image` command for manual triggering
- Auto-converts clipboard image formats to PNG

## Requirements

- WSLg clipboard support (`wl-paste`)
- ImageMagick (`convert`)

## Install

```bash
pi install npm:@guwidoe/pi-wsl-clipboard-image
```
